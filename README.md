# Aryan | Software Engineer Portfolio

A premium, interactive personal portfolio website showcasing projects, skills, and my learning journey as a Computer Science student.

## 🚀 Live Demo
**[View Live Portfolio on Vercel](https://aryancodes352-portfolio.vercel.app/)**  
*(Note: Replace this link with your exact URL once Vercel finishes deploying!)*

## ✨ Features
- **Modern Aesthetic:** Dark mode "Cyber/Matrix" layout with glassmorphic elements and high-end typography.
- **Interactive Particle Network:** A fully integrated `<canvas>` background that responds parallaxally to mouse physics.
- **Scroll Storytelling:** A glowing vertical timeline that tracks scroll progress dynamically.
- **3D Tilt Cards:** Premium interactive hovering using Framer Motion to give substantial depth to layout components.
- **Bento Box Grid:** Minimalist, fully responsive Grid/Flex layouts.

## 💻 Tech Stack
- Frontend: Next.js 16 (App Router), React 19
- Styling: Tailwind CSS v4
- Animations: Framer Motion
- Icons: Lucide React

## 🛠️ Local Development
First, install the dependencies if you haven't:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Deployment via Vercel
The code is directly deployable to [Vercel](https://vercel.com/new).
Simply log into Vercel, import your `AryanCodes352/portfolio` GitHub repository, and click **Deploy**. Vercel will automatically detect the Next.js setup and run the build.
